📝 Kanban Board – Task Management Web App

A simple and interactive Kanban Task Board built using HTML, CSS, and JavaScript.
This project allows users to visually manage tasks across three stages — To Do, In Progress, and Done — using an intuitive drag-and-drop interface. All tasks are stored in LocalStorage, so your data stays safe even after refreshing the page.

⸻

🚀 Features
	•	Add New Tasks
Users can create tasks with a title and description through a clean modal popup.
	•	Drag & Drop Support
Move tasks between columns (To Do → In Progress → Done) smoothly using the native JavaScript drag-and-drop API.
	•	Delete Tasks
Remove tasks instantly with a delete button.
	•	Light & Dark Mode
A toggle button allows switching themes. The selected theme is saved in LocalStorage.
	•	Persistent Data Storage
All tasks are saved in LocalStorage, ensuring they remain after page reloads.
